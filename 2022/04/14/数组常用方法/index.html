数组常用方法
2022-04-14 02:13:35 # JavaScript

  1. arr.push() / arr.pop()

    向数组的末尾添加/删除一个或多个元素，并返回新的数组长度/弹出的元素（_数组为空返回 undefined_），原数组改变。可利用此两种方法模拟栈

  2. arr.unshift() / arr.shift()

    向数组的开头添加/删除一个或多个元素，并返回新的数组长度/弹出的元素（_数组为空返回 undefined_），原数组改变。可利用此两种方法模拟队列（_通常结合 push()_）

  3. arr.concat()

    合并两个或多个数组，生成一个新的数组。原数组不变

  4. arr.join()

    将数组的每一项用指定字符连接形成一个字符串。默认连接字符为 “,” 英文逗号

  5. arr.reverse()

    将数组倒序，返回翻转后的数组，原数组改变

  6. arr.sort()

    对数组元素按照字符串 UniCode 码排序（_可传入自定义函数，返回负值代表前者位置应该在后者前面_），返回排序后的数组，原数组改变

  7. arr.forEach()

    调用数组的每个元素，并将元素传递给回调函数，原数组不变（_注意该方法和 map 的区别，forEach 不返回新数组，返回 undefined_）

  8. arr.map()

    原数组的每一项执行函数后，返回一个新的数组，原数组不变

  9. arr.filter()

    过滤数组中，符合条件的元素并返回一个新的数组，原数组不变

  10. arr.every()

    对数组中的每一项进行判断，若都符合则返回 true，否则返回 false

  11. arr.some()

    对数组中的每一项进行判断，若都不符合则返回 false，否则返回 true

  12. arr.reduce()

    接收一个函数作为累加器，数组中的每个值（从左到右）开始缩减，最终计算为一个值

    回调函数有四个参数，分别是：

    • prev 上一次调用回调返回的值，第一次遍历时该 prev 是数组的第一项;
    • curr 是数组中当前被处理的每一项元素，从数组的第二项开始;
    • index 当前被处理元素在数组中的索引，从 1 开始;
    • array 调用 reduce 方法的原数组。

    该回调函数被调用 length-1 次

  13. arr.includes()

    判断一个数组是否包含一个指定的值，如果是返回 true，否则 false，接收两个参数，第一个参数是需要查找的元素值，第二个是指定从何处开始索引，默认为 0，若为负数从 arr.length + fromIndex 处开始查找

    字符串也有该方法，判断字符串是否包含指定的子字符串，如果找到匹配的字符串则返回 true，否则返回 false，接收参数与含义与数组相同

  14. arr.splice(index,howmany,item1, …, itemX)

    splice() 方法向/从数组中添加/删除项目，然后返回被删除的项目,该方法会改变原始数组

  15. arr.slice(start,end)

    可从已有的数组中返回选定的元素
